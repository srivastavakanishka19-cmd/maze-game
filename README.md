# 🧩 Python Maze Escape

A clean, interactive maze exploration game built with Python. Navigate through a complex labyrinth, test your logic, and reach the exit in record time!

## 🚀 Features
* **Keyboard Controls:** Smooth movement using Arrow Keys or WASD.
* **Collision Detection:** Intelligent logic to prevent walking through walls.
* **Dynamic Graphics:** Built using the `pygame` library for a responsive experience.
* **Win Condition:** Automated detection when the player reaches the goal.

## 🛠️ Installation

### 1. Prerequisites
Ensure you have **Python 3.x** installed. You will also need the `pygame` library.
